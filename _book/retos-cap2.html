2.1 Reto 2.1: Quiero pizza

CautionDificultad

Básico

¿Qué imprime este código?

console.log("I want pizza"[0])

Opciones:

  • A. """
  • B. "I"
  • C. SyntaxError
  • D. undefined
TipPista

Las cadenas, al igual que los arreglos, son elementos iterables.

2.2 Reto 2.2: Interpolación de cadenas

CautionDificultad

Básico

¿Qué imprime este código?

console.log(`${(x => x)('I love')} to program`)

Opciones:

  • A. I love to program
  • B. undefined to program
  • C. ${(x => x)('I love') to program
  • D. TypeError
TipPista

En las plantillas con backticks, todo lo que este dentro de {} se evalua como una expresión de JavaScript.

2.12 Reto 2.12: El método repeat de las cadenas

Intermedio

console.log("tea" + ".".repeat(5) + ":" + "$1.50");

console.log("beer" + ".".repeat(-1) + ":" + "$5.00");

Opciones:

--- Menu ---
tea.....:$1.50
coffee..:$3.75
RangeError: repeat count must be non-negative
--- Menu ---
tea.....:$1.50
coffee..:$3.75
beer....:$5.00
--- Menú ---
té.....:$1.50
RangeError: repeat count must be non-decimal numbers
RangeError: repeat count must be non-negative numbers
TipPista

repeat es un método de cadenas poco conocido pero muy útil para repetir cadenas de manera controlada.
