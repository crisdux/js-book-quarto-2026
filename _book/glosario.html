Glosario

Definiciones de términos técnicos utilizados en este libro para facilitar la comprensión de conceptos clave de JavaScript.

A

Asignación por referencia

Inglés: Assignment by reference

Tipo: Concepto

Definición:

Proceso mediante el cual se copia la dirección de memoria de una variable a otra. En JavaScript, esto ocurre con los tipos de datos complejos (como objetos y arreglos). Al realizar la asignación, ambas variables apuntan al mismo objeto en memoria, por lo que cualquier modificación en uno de los objetos afectará a ambos.

Ejemplo:

let obj1 = { a: 1 };
let obj2 = obj1;
obj2.a = 2;
console.log(obj1.a); // 2

Referencias: Montero (2025)

Asignación por valor

Inglés: Assignment by value

Tipo: Concepto

Definición:

Proceso mediante el cual se copia el contenido real de una variable a otra. En JavaScript, esto ocurre con los tipos de datos primitivos (como números, cadenas de texto y booleanos). Al realizar la asignación, se crea una copia independiente en memoria, por lo que cualquier modificación posterior en una de las variables no afectará a la otra.

Ejemplo:

let a = 1;
let b = a;
b = 2;
console.log(a); // 1

Referencias: Montero (2025)

Array.from

Inglés: Array.from

Tipo: Método estático

Definición:

El método Array.from() crea una nueva instancia de Array a partir de un objeto iterable.

Ejemplo:

console.log(Array.from("JavaScript"));
// [ 'J', 'a', 'v', 'a', 'S', 'c', 'r', 'i', 'p', 't' ]

Referencias: Mozilla Developer Network (2025b)

Ámbito de bloque

Inglés: Block scope

Tipo: Concepto

Definición:

Ámbito de bloque es un concepto que se refiere a la visibilidad y accesibilidad de las variables dentro de un bloque de código, como un bucle o una condición.

Ejemplo:

if (true) {
    let x = 1;
}
console.log(x); // ReferenceError: x is not defined

Referencias: Cardillo (2025)

Ámbito de función

Inglés: Function scope

Tipo: Concepto

Definición:

Ámbito de función es un concepto que se refiere a la visibilidad y accesibilidad de las variables dentro de una función.

Ejemplo:

function miFuncion() {
    var x = 10;
}
console.log(x); // ReferenceError: x is not defined

Referencias: Aprende JavaScript (2025)

Arreglo

Inglés: Array

Tipo: Objeto

Definición:

Una colección ordenada de valores donde cada elemento tiene una posición numérica denominada índice. En JavaScript, los arreglos son objetos globales especializados que permiten almacenar múltiples elementos bajo un solo nombre de variable, son dinámicos y pueden contener datos de diferentes tipos simultáneamente.

Ejemplo:

const myArray = ["JavaScript", 2026, true];
console.log(myArray[0]); // "JavaScript"

Referencias: Mozilla Developer Network (2025a)

at

Inglés: at

Tipo: Método

Definición: Método que retorna el elemento en la posición especificada de un arreglo o cadena.

Ejemplo:

const myArray = ["JavaScript", 2026, true];
console.log(myArray.at(0)); // "JavaScript"

Referencias: MDN Web Docs (2025b)

B

Base binaria

Inglés: Binary

Tipo: Concepto

Definición:

Sistema numérico que utiliza solo dos símbolos para representar valores: 0 y 1.

Ejemplo:

let binario = 0b1010; // Representa el número 10 en decimal
console.log(binario); // 10

Referencias: Loor Delgado and Trejos Buriticá (2018)

Base octal

Inglés: Octal

Tipo: Concepto

Definición: Sistema numérico que utiliza ocho símbolos para representar valores: 0, 1, 2, 3, 4, 5, 6 y 7.

Ejemplo:

let octal = 0o12; // Representa el número 10 en decimal
console.log(octal); // 10

Referencias: Cuemath (2025)

Base decimal

Inglés: Decimal

Tipo: Concepto

Definición: Sistema numérico que utiliza diez símbolos para representar valores: 0, 1, 2, 3, 4, 5, 6, 7, 8 y 9.

Ejemplo:

let decimal = 10; // Representa el número 10 en decimal
console.log(decimal); // 10

Referencias: Encyclopædia Britannica (2025)

C

Cadena de caracteres

Inglés: String

Tipo: Primitivo

Definición:

Tipo de dato primitivo que representa una secuencia inmutable de caracteres utilizada para representar texto. En JavaScript, las cadenas se almacenan como una serie de unidades de código de 16 bits siguiendo el estándar UTF-16, donde cada elemento ocupa una posición indexada comenzando desde cero.

Ejemplo:

let cadena = "Hola, mundo!";
console.log(cadena.length); // 13

Referencias: Mozilla Developer Network (2025r)

Call Stack

Inglés: Call Stack

Tipo: Concepto

Definición:

Estructura de datos de tipo LIFO (Last In, First Out) que JavaScript utiliza para rastrear la ejecución de las funciones. Cuando se invoca una función, esta se coloca en la parte superior de la pila (push), y cuando la función termina de ejecutarse, el motor de JavaScript la quita de la pila (pop) para continuar con la ejecución donde se quedó.

Ejemplo:

function greet() {
  console.log("Hello!");
}

function speak() {
  greet();
}

speak(); 
// The stack follows the order: speak -> greet -> console.log
// Then it is emptied in reverse order.

Referencias: MDN Web Docs (2026b)

Coerción de tipos

Inglés: Type Coercion

Tipo: Concepto

Definición:
Conversión automática o implícita de valores de un tipo de dato a otro realizada por el intérprete de JavaScript.

Ejemplo:

"5" + 3   // "53" - convierte number a string
"5" - 3   // 2 - convierte string a number

Referencias: Mozilla Developer Network (2024b)

concat

Inglés: concat

Tipo: Método

Definición: Método que concatena dos o más arreglos en uno nuevo.

Ejemplo:

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];
const array3 = array1.concat(array2);
console.log(array3); // [1, 2, 3, 4, 5, 6]

Referencias: MDN Web Docs (2025c)

Ejemplo:

console.count("foo"); // foo: 2
console.count("bar"); // bar: 1

Ejemplo:

let string1 = "Hello";

const

Inglés: const

Definición: Declaración de una variable que no puede ser reasignada.

Ejemplo:

Referencias: MDN Web Docs (2025j)

Corto circuito

Inglés: Short-circuit

Tipo: Concepto

Definición: Comportamiento de los operadores lógicos en el que el segundo operando es evaluado solo si el primero no es suficiente para determinar el resultado de la expresión.

Ejemplo:

"Usuario" || "Invitado"; // "Usuario"
0 && "No se evalúa"; // 0

Referencias: GeeksforGeeks (2025c)

D

delete

Inglés: delete

Tipo: Operador

Definición: Operador que elimina una propiedad de un objeto. Si la propiedad se elimina con éxito, devuelve true; de lo contrario, devuelve false. Es importante notar que no libera memoria directamente, sino que rompe la conexión entre el objeto y la propiedad.

Ejemplo:

const user = { name: "Alice", role: "Developer" };

delete user.role;
console.log(user); // { name: "Alice" }
console.log("role" in user); // false

Referencias: MDN Web Docs (2026c)

Desestructuración

Inglés: Destructuring

Tipo: Concepto

Definición:

Sintaxis que permite extraer valores de arreglos o propiedades de objetos y asignarlos a variables individuales de forma compacta. En el caso de los arreglos, se utiliza la posición de los elementos para determinar qué valor se asigna a cada variable.

Ejemplo:

// Array destructuring
const fruits = ["apple", "pear", "grape"];
const [first, second] = fruits;
console.log(first);  // "apple"
console.log(second); // "pear"

Referencias: duxtech (2021)

Doble negación

Inglés: Double Negation

Tipo: Concepto

Definición: Uso de dos operadores de negación lógica (!) consecutivos para convertir un valor de cualquier tipo a su valor booleano explícito. El primer ! convierte el valor a su opuesto booleano y el segundo lo invierte de nuevo, devolviendo true para valores truthy y false para valores falsy.

Ejemplo:

!!"hola"  // true
!!0       // false
!!null    // false

Referencias: Mozilla Developer Network (2024b)

E

entries

Inglés: entries

Tipo: Método

Definición: Método que devuelve un nuevo objeto iterador que contiene los pares clave/valor para cada índice en el arreglo.

Ejemplo:

const arr = ["a", "b", "c"];
const iterator = arr.entries();

for (const [index, element] of iterator) {
  console.log(index, element);
}
// 0 "a"
// 1 "b"
// 2 "c"

Referencias: MDN Web Docs (2026l)

ES6

Inglés: ES6

Tipo: Concepto

Definición: Sexta edición del estándar ECMAScript publicada en 2015. Representa una de las actualizaciones más significativas del lenguaje JavaScript, introduciendo nuevas características y sintaxis que modernizaron el desarrollo, como las declaraciones let y const, funciones de flecha, clases, plantillas de cadena y módulos.

Ejemplo:

// Uso de arrow functions y template literals (características de ES6)
const saludar = (nombre) => `Hola, ${nombre}!`;
console.log(saludar("JavaScript")); // "Hola, JavaScript!"

Referencias: W3Schools (2025)

Event Loop

Inglés: Event Loop

Tipo: Concepto

Definición: Mecanismo fundamental en el entorno de ejecución de JavaScript que permite realizar operaciones no bloqueantes a pesar de ser un lenguaje de un solo hilo. Su función principal es monitorear constantemente la pila de llamadas (Call Stack) y la cola de tareas (Callback Queue); cuando la pila de llamadas se encuentra vacía, el Event Loop toma la primera tarea de la cola y la traslada a la pila para su ejecución.

Ejemplo

console.log("Start");

setTimeout(() => {
  console.log("Inside setTimeout callback");
}, 0);

console.log("End");

// Output:
// Start
// End
// Inside setTimeout callback

Referencias: MDN Web Docs (2026i)

Expresión

Inglés: Expression

Tipo: Concepto

Definición: Cualquier unidad de código válida que se resuelve en un valor. En JavaScript, las expresiones pueden consistir en valores literales, variables, operadores o llamadas a funciones que el intérprete evalúa para producir un resultado único.

Ejemplo:

console.log(5 + 3);          // Expresión aritmética
console.log("Hello" + "!");  // Expresión de cadena
console.log(true && false);  // Expresión lógica

Referencias: Mozilla Developer Network (2025c)

F

Falsy

Inglés: Falsy

Tipo: Concepto

Definición: Valor que se convierte en false cuando se evalúa en un contexto booleano.

Ejemplo:

Boolean(0);         // false
Boolean("");        // false
Boolean(null);      // false
Boolean(undefined); // false
Boolean(NaN);       // false

Referencias: Mozilla Developer Network (2025d)

flat

Inglés: flat

Tipo: Método

Definición: Método que crea un nuevo arreglo con los elementos concatenados recursivamente hasta una profundidad especificada. Dicho eb otras palabras, permite “aplanar” un arreglo anidado un número determinado de veces.

Ejemplo:

const arr = [1, 2, [3, 4, [5]]];
const flatArr = arr.flat(2); // [1, 2, 3, 4, 5]
console.log(flatArr);

Referencias: MDN Web Docs (2025d)

Función

Inglés: Function

Tipo: Objeto

Definición: Función es una unidad de código que puede ser invocada para realizar una tarea específica. En JavaScript, las funciones son ciudadanos de primera clase, lo que significa que pueden ser asignadas a variables, pasadas como argumentos a otras funciones y retornadas como valores.

Ejemplo:

function myFunction() {
    console.log("Hello, world!");
}

Referencias: MDN Web Docs (2025m)

Función flecha

Inglés: Arrow Function

Tipo: Concepto

Definición: Sintaxis compacta para escribir expresiones de funciones en JavaScript, introducida en ES6. Se caracteriza por omitir la palabra clave function y utilizar el operador =>. A diferencia de las funciones tradicionales, no posee sus propios enlaces para this, arguments, super o new.target, heredando estos valores del contexto léxico circundante.

Ejemplo:

const add = (a, b) => a + b;

Referencias: MDN Web Docs (2026a)

Función anónima

Inglés: Anonymous Function

Tipo: Concepto

Definición: Función que no tiene nombre y se puede asignar a una variable o pasar como argumento a otra función.

Ejemplo:

const miFuncion = function() {
    console.log("Hola, mundo!");
};

Referencias: GeeksforGeeks (2025b)

for…of

Inglés: for…of

Tipo: Sentencia

Definición: Sentencia que itera sobre los elementos de un iterable.

Ejemplo:

const array = [1, 2, 3];
for (const item of array) {
    console.log(item);
}
// 1
// 2
// 3

Referencias: MDN Web Docs (2025l)

for…in

Inglés: for…in

Tipo: Sentencia

Definición: Sentencia que itera sobre las propiedades de un objeto.

Ejemplo:

const user = {
  name: "Alice",
  role: "Developer",
  country: "Bolivia"
};

for (const key in user) {
  console.log(`${key}: ${user[key]}`);
}
// name: Alice
// role: Developer
// country: Bolivia

Referencias: MDN Web Docs (2025k)

G

Global

Inglés: Global

Tipo: Concepto

Definición: Objeto que reside en el ámbito de mayor nivel y es accesible desde cualquier parte de la aplicación sin necesidad de declaraciones explícitas. En Node.js, este objeto se denomina global y proporciona acceso a variables y funciones del sistema (como process o __dirname). El estándar moderno globalThis unifica el acceso a este objeto global independientemente de si el código se ejecuta en un servidor o en un navegador.

Ejemplo:

console.log(globalThis); // Global object

Referencias: MDN Web Docs (2025n)

H

hasOwnProperty

Inglés: hasOwnProperty

Tipo: Método

Definición:

Método que devuelve un booleano indicando si el objeto tiene la propiedad especificada como una propiedad propia (no heredada). Es una herramienta fundamental para verificar la existencia de claves en un objeto antes de intentar acceder a ellas, evitando recorrer propiedades del prototipo.

Ejemplo:

const user = { name: "Alice", role: "Developer" };
console.log(user.hasOwnProperty("name")); // true
console.log(user.hasOwnProperty("toString")); // false (es heredada)

Referencias: MDN Web Docs (2026m)

Higher Order Function

Inglés: Higher Order Function

Tipo: Concepto

Definición: Función que recibe como parámetro otra función o retorna una función.

Ejemplo:

function higherOrderFunction(callback) {
  callback();
}

higherOrderFunction(() => console.log("Hello!"));

Referencias: freeCodeCamp (2026)

Hoisting

Inglés: Hoisting

Tipo: Concepto

Definición: Comportamiento de JavaScript en el que las declaraciones de variables, funciones, clases o importaciones se mueven conceptualmente a la parte superior de su ámbito (scope) durante la fase de compilación, antes de que se ejecute el código. Esto permite que ciertos elementos sean referenciados antes de su declaración formal, aunque su disponibilidad e inicialización dependen de la forma en que fueron declarados.

Ejemplo:

console.log(x); // ReferenceError: x is not defined
let x = 1;

Referencias: freeCodeCamp (2025a)

I

Igualdad débil

Inglés: Weak Equality

Tipo: Concepto

Definición: Operador que compara dos valores para verificar si son iguales, convirtiendo los valores si es necesario para que tengan el mismo tipo.

Ejemplo:

console.log(1 == "1"); // true
console.log(true == 1); // true

Referencias: MDN Web Docs (2025h)

Igualdad estricta

Inglés: Strict Equality

Tipo: Concepto

Definición: Operador que compara dos valores para verificar si son iguales y del mismo tipo.

Ejemplo:

console.log(1 === 1);   // true
console.log(1 === "1"); // false

Referencias: Mozilla Developer Network (2025e)

in

Inglés: in

Tipo: Operador

Definición: Operador que devuelve true si la propiedad especificada existe en el objeto (o en su cadena de prototipos). Se utiliza frecuentemente para verificar la existencia de una clave en un objeto antes de intentar acceder a su valor.

Ejemplo:

const user = { name: "Alice", role: "Developer" };

console.log("name" in user);   // true
console.log("salary" in user); // false
console.log("toString" in user); // true (propiedad heredada)

Referencias: MDN Web Docs (2026d)

Interprete de JavaScript

Inglés: JavaScript Interpreter

Tipo: Concepto

Definición: Programa que procesa y ejecuta el código fuente de JavaScript directamente, traduciéndolo a instrucciones de máquina en tiempo de ejecución. A diferencia de los lenguajes compilados, no requiere una fase de traducción previa a un archivo binario ejecutable.

Ejemplo:

console.log("The interpreter processes this line");

Referencias: GeeksforGeeks (2025a)

Referencias: MDN Web Docs (2025a)

Iterador

Inglés: Iterator

Tipo: Concepto

Definición: Objeto que permite recorrer los elementos de una colección uno a uno. En JavaScript, las cadenas de texto son iterables, lo que permite acceder a cada uno de sus caracteres de forma secuencial mediante estructuras de control como bucles o a través de su índice.

Ejemplo:

  console.log(char);
}
// "H"
// "e"
// "l"
// "l"
// "o"
