Glosario

Definiciones de términos técnicos utilizados en este libro para facilitar la comprensión de conceptos clave de JavaScript.

A

Array.from

Inglés: Array.from

Tipo: Método estático

Definición:

El método Array.from() crea una nueva instancia de Array a partir de un objeto iterable.

Ejemplo:

console.log(Array.from("JavaScript"));
// [ 'J', 'a', 'v', 'a', 'S', 'c', 'r', 'i', 'p', 't' ]

Referencias: Mozilla Developer Network (2025b)

Ámbito de bloque

Inglés: Block scope

Tipo: Concepto

Definición:

Ámbito de bloque es un concepto que se refiere a la visibilidad y accesibilidad de las variables dentro de un bloque de código, como un bucle o una condición.

Ejemplo:

if (true) {
    let x = 1;
}
console.log(x); // ReferenceError: x is not defined

Referencias: Cardillo (2025)

Ámbito de función

Inglés: Function scope

Tipo: Concepto

Definición:

Ámbito de función es un concepto que se refiere a la visibilidad y accesibilidad de las variables dentro de una función.

Ejemplo:

function miFuncion() {
    var x = 10;
}
console.log(x); // ReferenceError: x is not defined

Referencias: Aprende JavaScript (2025)

Arreglo

Inglés: Array

Tipo: Objeto

Definición:

Una colección ordenada de valores donde cada elemento tiene una posición numérica denominada índice. En JavaScript, los arreglos son objetos globales especializados que permiten almacenar múltiples elementos bajo un solo nombre de variable, son dinámicos y pueden contener datos de diferentes tipos simultáneamente.

Ejemplo:

const miArreglo = ["JavaScript", 2024, true];
console.log(miArreglo[0]); // "JavaScript"

Referencias: Mozilla Developer Network (2025a)

B

Base binaria

Inglés: Binary

Tipo: Concepto

Definición:

Sistema numérico que utiliza solo dos símbolos para representar valores: 0 y 1.

Ejemplo:

let binario = 0b1010; // Representa el número 10 en decimal
console.log(binario); // 10

Referencias: Loor Delgado and Trejos Buriticá (2018)

Base octal

Inglés: Octal

Tipo: Concepto

Definición: Sistema numérico que utiliza ocho símbolos para representar valores: 0, 1, 2, 3, 4, 5, 6 y 7.

Ejemplo:

let octal = 0o12; // Representa el número 10 en decimal
console.log(octal); // 10

Referencias: Cuemath (2025)

Base decimal

Inglés: Decimal

Tipo: Concepto

Definición: Sistema numérico que utiliza diez símbolos para representar valores: 0, 1, 2, 3, 4, 5, 6, 7, 8 y 9.

Ejemplo:

let decimal = 10; // Representa el número 10 en decimal
console.log(decimal); // 10

Referencias: Encyclopædia Britannica (2025)

C

Cadena de caracteres

Inglés: String

Tipo: Primitivo

Definición:

Tipo de dato primitivo que representa una secuencia inmutable de caracteres utilizada para representar texto. En JavaScript, las cadenas se almacenan como una serie de unidades de código de 16 bits siguiendo el estándar UTF-16, donde cada elemento ocupa una posición indexada comenzando desde cero.

Ejemplo:

let cadena = "Hola, mundo!";
console.log(cadena.length); // 13

Referencias: Mozilla Developer Network (2025r)

Coerción de tipos

Inglés: Type Coercion

Tipo: Concepto

Definición:
Conversión automática o implícita de valores de un tipo de dato a otro realizada por el intérprete de JavaScript.

Ejemplo:

"5" + 3   // "53" - convierte number a string
"5" - 3   // 2 - convierte string a number

Referencias: Mozilla Developer Network (2024b)

Concatenación

Inglés: Concatenation

Tipo: Concepto

Definición: Operación que une dos cadenas de texto para formar una nueva cadena.

Ejemplo:

let string1 = "Hello";
let string2 = "world";
let result = string1 + ", " + string2; // "Hello, world"

Referencias: Mozilla Developer Network (2025s)

const

Inglés: const

Tipo: Concepto

Definición: Declaración de una variable que no puede ser reasignada.

Ejemplo:

const a = 1;
a = 2; // Error

Referencias: MDN Web Docs (2025b)

Corto circuito

Inglés: Short-circuit

Tipo: Concepto

Definición: Comportamiento de los operadores lógicos en el que el segundo operando es evaluado solo si el primero no es suficiente para determinar el resultado de la expresión.

Ejemplo:

"Usuario" || "Invitado"; // "Usuario"
0 && "No se evalúa"; // 0

Referencias: GeeksforGeeks (2025c)

D

Doble negación

Inglés: Double Negation

Tipo: Concepto

Definición: Uso de dos operadores de negación lógica (!) consecutivos para convertir un valor de cualquier tipo a su valor booleano explícito. El primer ! convierte el valor a su opuesto booleano y el segundo lo invierte de nuevo, devolviendo true para valores truthy y false para valores falsy.

Ejemplo:

!!"hola"  // true
!!0       // false
!!null    // false

Referencias: Mozilla Developer Network (2024b)

E

Expresión

Inglés: Expression

Tipo: Concepto

Definición: Cualquier unidad de código válida que se resuelve en un valor. En JavaScript, las expresiones pueden consistir en valores literales, variables, operadores o llamadas a funciones que el intérprete evalúa para producir un resultado único.

Ejemplo:

console.log(5 + 3);          // Expresión aritmética
console.log("Hello" + "!");  // Expresión de cadena
console.log(true && false);  // Expresión lógica

Referencias: Mozilla Developer Network (2025c)

ES6

Inglés: ES6

Tipo: Concepto

Definición: Sexta edición del estándar ECMAScript publicada en 2015. Representa una de las actualizaciones más significativas del lenguaje JavaScript, introduciendo nuevas características y sintaxis que modernizaron el desarrollo, como las declaraciones let y const, funciones de flecha, clases, plantillas de cadena y módulos.

Ejemplo:

// Uso de arrow functions y template literals (características de ES6)
const saludar = (nombre) => `Hola, ${nombre}!`;
console.log(saludar("JavaScript")); // "Hola, JavaScript!"

Referencias: W3Schools (2025)

F

Falsy

Inglés: Falsy

Tipo: Concepto

Definición: Valor que se convierte en false cuando se evalúa en un contexto booleano.

Ejemplo:

Boolean(0);         // false
Boolean("");        // false
Boolean(null);      // false
Boolean(undefined); // false
Boolean(NaN);       // false

Referencias: Mozilla Developer Network (2025d)

Función

Inglés: Function

Tipo: Objeto

Definición: Función es una unidad de código que puede ser invocada para realizar una tarea específica. En JavaScript, las funciones son ciudadanos de primera clase, lo que significa que pueden ser asignadas a variables, pasadas como argumentos a otras funciones y retornadas como valores.

Ejemplo:

function myFunction() {
    console.log("Hello, world!");
}

Referencias: MDN Web Docs (2025c)

Función anónima

Inglés: Anonymous Function

Tipo: Concepto

Definición: Función que no tiene nombre y se puede asignar a una variable o pasar como argumento a otra función.

Ejemplo:

const miFuncion = function() {
    console.log("Hola, mundo!");
};

Referencias: GeeksforGeeks (2025b)

G

Global

Inglés: Global

Tipo: Concepto

Definición: Objeto que reside en el ámbito de mayor nivel y es accesible desde cualquier parte de la aplicación sin necesidad de declaraciones explícitas. En Node.js, este objeto se denomina global y proporciona acceso a variables y funciones del sistema (como process o __dirname). El estándar moderno globalThis unifica el acceso a este objeto global independientemente de si el código se ejecuta en un servidor o en un navegador.

Ejemplo:

console.log(globalThis); // Global object

Referencias: MDN Web Docs (2025d)

I

Igualdad débil

Inglés: Weak Equality

Tipo: Concepto

Definición: Operador que compara dos valores para verificar si son iguales, convirtiendo los valores si es necesario para que tengan el mismo tipo.

Ejemplo:

console.log(1 == "1"); // true
console.log(true == 1); // true

Referencias: MDN Web Docs (2025a)

Igualdad estricta

Inglés: Strict Equality

Tipo: Concepto

Definición: Operador que compara dos valores para verificar si son iguales y del mismo tipo.

Ejemplo:

console.log(1 === 1);   // true
console.log(1 === "1"); // false

Referencias: Mozilla Developer Network (2025e)

Interprete de JavaScript

Inglés: JavaScript Interpreter

Tipo: Concepto

Definición: Programa que procesa y ejecuta el código fuente de JavaScript directamente, traduciéndolo a instrucciones de máquina en tiempo de ejecución. A diferencia de los lenguajes compilados, no requiere una fase de traducción previa a un archivo binario ejecutable.

Ejemplo:

console.log("The interpreter processes this line");

Referencias: GeeksforGeeks (2025a)

Iterador

Inglés: Iterator

Tipo: Concepto

Definición: Objeto que permite recorrer los elementos de una colección uno a uno. En JavaScript, las cadenas de texto son iterables, lo que permite acceder a cada uno de sus caracteres de forma secuencial mediante estructuras de control como bucles o a través de su índice.

Ejemplo:

const text = "Hello";

for (const char of text) {
  console.log(char);
}
// "H"
// "e"
// "l"
// "l"
// "o"

Referencias: Mozilla Developer Network (2025f)

J

Java

Inglés: Java

Tipo: Lenguaje de programación

Definición: Lenguaje de programación orientado a objetos que se utiliza para crear aplicaciones empresariales, aplicaciones móviles, aplicaciones del lado del servidor, etc.

Ejemplo:

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
    }
}

Referencias: Microsoft Azure (2025)

JavaScript

Inglés: JavaScript

Tipo: Lenguaje de programación

Definición: Lenguaje de programación interpretado, dinámico y orientado a prototipos que se utiliza para crear aplicaciones web interactivas, aplicaciones móviles, aplicaciones de escritorio, aplicaciones del lado del servidor, etc.

Ejemplo:

console.log("Hola, mundo!");

Referencias: Mozilla Developer Network (2025g)

join

Inglés: join

Tipo: Método

Definición: Método que convierte un arreglo a cadena de texto mediante un separador pasado como parámetro.

Ejemplo:

let result = ["Hello", "JavaScript"].join(" ");
console.log(result); // "Hello JavaScript"

Referencias: Percival, Svekis, and Putten (2021)

JSON

Inglés: JSON

Tipo: Objeto

Definición: Formato ligero de intercambio de datos, basado en texto y fácil de leer para los humanos. Aunque se deriva de la sintaxis de objetos de JavaScript, es un formato independiente del lenguaje y es ampliamente utilizado para almacenar y transmitir datos estructurados entre un servidor y una aplicación web o entre diferentes sistemas.

Ejemplo:

{
  "nombre": "Juan Pérez",
  "edad": 30,
  "esEstudiante": false,
  "hobbies": ["lectura", "programación"],
  "direccion": {
    "calle": "Calle Benito de los Palotes 123",
    "ciudad": "Springfield"
  }
}

Referencias: Mozilla Developer Network (2025v)

JSON.stringify

Inglés: JSON.stringify

Tipo: Método estático

Definición: Método estático que convierte un valor de JavaScript en una cadena JSON.

Ejemplo:

JSON.stringify({ a: 1, b: 2 }) // "{"a":1,"b":2}"

Referencias: Mozilla Developer Network (2025h)

L

length

Inglés: length

Tipo: Propiedad

Definición: Propiedad que indica la longitud de una cadena de texto o un arreglo.

Ejemplo:

console.log("hello".length); // 4
console.log([1, 2, 3].length); // 3

Referencias: W3Schools (2026)

let

Inglés: let

Tipo: Operador

Definición: Operador que crea una variable con ámbito de bloque.

Ejemplo:

let a = 1;
let b = 2;

Referencias: Mozilla Developer Network (2025i)

N

NaN

Inglés: NaN

Tipo: Primitivo

Definición: Valor especial que representa un número no válido.

Ejemplo:

0 / 0 // NaN
"hola" * 2 // NaN

Referencias: Mozilla Developer Network (2025j)

new

Inglés: new

Tipo: Operador

Definición: Operador que crea una instancia de un objeto.

Ejemplo:

const a = new Number(2);

Referencias: Mozilla Developer Network (2025k)

Node.js

Inglés: Node.js

Tipo: Entorno de ejecución

Definición: Entorno de ejecución que permite ejecutar JavaScript fuera del navegador, utilizando el motor V8 de Google Chrome.

Ejemplo:

console.log("Hello Node.js");

Referencias: MDN Web Docs (2025e)

Notación de corchetes

Inglés: Bracket Notation

Tipo: Concepto

Definición: Sintaxis que permite acceder a las propiedades de un objeto o a los elementos de un arreglo utilizando corchetes []. En los objetos, esta notación permite emplear cadenas de texto para referenciar claves, lo que resulta esencial cuando el nombre de la propiedad es dinámico, contiene espacios, caracteres especiales o comienza con un número. En el caso de los arreglos, se utiliza para acceder a sus elementos mediante su índice numérico posicional.

Ejemplo:

const persona = {
  nombre: "Juan",
  edad: 30
};

console.log(persona["nombre"]); // "Juan"
console.log(persona["edad"]); // 30

Referencias: freeCodeCamp Español (2024)

null

Inglés: null

Tipo: Primitivo

Definición: Valor especial que representa la ausencia de valor o un objeto que no existe.

Ejemplo:

let usuario = null; // Representa la ausencia intencional de un objeto
console.log(usuario); // null

Referencias: Mozilla Developer Network (2025l)

Number

Inglés: Number

Tipo: Objeto

Definición:
Objeto global que proporciona propiedades y métodos para trabajar con números.

Ejemplo:

Number.parseInt("5") // 5

Referencias: Mozilla Developer Network (2024a)

Nullish coalescing operator

Inglés: Nullish coalescing operator

Tipo: Operador

Definición: Operador que evalúa dos valores y regresa el primero si es nullish (es decir, null o undefined), de lo contrario regresa el segundo valor.

Ejemplo:

const a = null ?? "Valor por defecto"; // "Valor por defecto"
const b = "Valor" ?? "Valor por defecto"; // "Valor"

Referencias: JavaScript.info (2025)

O

Object.keys

Inglés: Object.keys

Tipo: Método estático

Definición: Método estático que regresa un arreglo con las claves (propiedades) de un objeto.

Ejemplo:

Object.keys({ a: 1, b: 2 }) // ["a", "b"]

Referencias: Mozilla Developer Network (2025n)

P

parseInt

Inglés: parseInt

Tipo: Función

Definición: Función que convierte un valor a tipo number de una base concreta (base binaria, base octal, base decimal, etc).

Ejemplo:

Number.parseInt("5") // 5

Referencias: Mozilla Developer Network (2025o)

R

repeat

Inglés: repeat

Tipo: Método

Definición: Método que concatena una cadena de texto un número determinado de veces.

Ejemplo:

console.info("hello".repeat(3)); // "hellohellohello"

Referencias: Rauschmayer (2021)

return

Inglés: return

Tipo: Operador

Definición: Operador que regresa un valor desde una función.

Ejemplo:

function sum(a, b) {
  return a + b;
}

Referencias: Mozilla Developer Network (2025q)

return implícito

Inglés: Return implicit

Tipo: Concepto

Definición: Comportamiento en el que una función devuelve un valor sin necesidad de utilizar explícitamente la palabra clave return. En JavaScript, esto ocurre principalmente en las funciones de flecha (arrow functions) cuando el cuerpo de la función consiste en una sola expresión.

Ejemplo:

const sumar = (a, b) => a + b;

Referencias: Demirel’s Organization (2025)

reverse

Inglés: reverse

Tipo: Método

Definición: Método que invierte el orden de los elementos de un arreglo.

Ejemplo:

console.info([1, 2, 3].reverse()); // [3, 2, 1]

Referencias: Rauschmayer (2021)

S

scope

Inglés: Scope

Tipo: Concepto

Definición: Contexto que determina la accesibilidad (visibilidad) de las variables, funciones y objetos en una parte específica del código durante el tiempo de ejecución.

Ejemplo:

let x = 10;
function myFunction() {
    let x = 20;
    console.log(x); // 20
}
console.log(x); // 10

Referencias: MDN Web Docs (2025f)

split

Inglés: split

Tipo: Método

Definición: Método que divide una cadena en un array de subcadenas, utilizando un separador como parámetro.

Ejemplo:

let result = "Hello JavaScript";
let arr_result = result.split(" ");
console.log(arr_result);
// [ 'Hello', 'JavaScript' ]

Referencias: Percival, Svekis, and Putten (2021)

Spread Operator

Inglés: Spread Operator

Tipo: Operador

Definición: Operador que permite expandir los elementos de un iterable (como un array o una cadena) en una lista de argumentos o en una nueva estructura de datos.

Ejemplo:

const array1 = [1, 2, 3];
const array2 = [...array1, 4, 5]; // [1, 2, 3, 4, 5]

Referencias: Antani (2016)

Symbol

Inglés: Symbol

Tipo: Objeto

Definición: Objeto global que proporciona propiedades y métodos para trabajar con símbolos. Muy útiles y comunes para crear propiedades privadas en objetos.

Ejemplo:

Symbol("foo") // Symbol("foo")

Referencias: Mozilla Developer Network (2025t)

T

Template Literals

Inglés: Template Literals

Tipo: Concepto

Definición: Literales de cadena que permiten expresiones incrustadas. Permiten el uso de cadenas multilínea y la interpolación de expresiones mediante el uso de comillas invertidas (backticks).

Ejemplo:

const nombre = "Mundo";
console.log(`Hola ${nombre}`); // "Hola Mundo"

Referencias: freeCodeCamp Español (2024)

Truthy

Inglés: Truthy

Tipo: Concepto

Definición: Valor que se convierte en true cuando se evalúa en un contexto booleano.

Ejemplo:

Boolean(1);         // true
Boolean("hola");    // true
Boolean({});        // true
Boolean([]);        // true
Boolean(function() {}); // true
Boolean(Infinity);  // true

Referencias: Mozilla Developer Network (2025w)

typeof

Inglés: typeof

Tipo: Operador

Definición: Operador que regresa una cadena con el tipo de dato de una variable.

Ejemplo:

typeof 5 // "number"
typeof "hola" // "string"
typeof true // "boolean"
typeof {} // "object"
typeof [] // "object"
typeof function() {} // "function"
typeof null // "object"
typeof undefined // "undefined"

Referencias: Mozilla Developer Network (2023)

U

undefined

Inglés: undefined

Tipo: Primitivo

Definición: Valor especial que representa la ausencia de valor o una variable que no ha sido inicializada.

Ejemplo:

let a;
console.log(a); // undefined

Referencias: Mozilla Developer Network (2025x)

V

var

Inglés: var

Tipo: Operador

Definición: Operador que crea una variable con ámbito de función.

Ejemplo:

var a = 1;

Referencias: Mozilla Developer Network (2025y)

Valor primitivo

Inglés: Primitive value

Tipo: Concepto

Definición: Dato que no es un objeto y no tiene métodos ni propiedades. JavaScript define siete tipos de datos primitivos: string, number, bigint, boolean, undefined, symbol y null. Estos valores son inmutables, lo que significa que no pueden ser modificados una vez creados.

Ejemplo:

5 // number
"hola" // string
true // boolean
null // null
undefined // undefined

Referencias: Mozilla Developer Network (2025p)

W

window

Inglés: window

Tipo: Objeto

Definición: Objeto global que representa la ventana del navegador.

Ejemplo:

console.log(window.innerWidth); // Ancho de la ventana
